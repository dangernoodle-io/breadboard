# bb_bqueue

<!-- BEGIN bbtool:brief -->
Blocking mailbox/MPSC queue — capacity==1 selects mailbox mode (overwrite/reset), capacity>1 selects bounded-MPSC mode (send/dropped); peek/receive/count/capacity work in both. Zero heap: a Kconfig-sized static instance pool.
<!-- END bbtool:brief -->
