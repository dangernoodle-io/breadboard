# bb_fsm

<!-- BEGIN bbtool:brief -->
Table-driven finite state machine primitive: consumer-owned rows (state, event, guard, action, next), entry/exit hooks, and a fixed-size timer-arm seam for the shell to reconstruct real OS timers from. Pure per-instance library -- no autoinit, no global state, no lock, embedded by value in the consumer's own struct.
<!-- END bbtool:brief -->
