# bb_core

<!-- BEGIN bbtool:brief -->
Foundational, near-zero-dep primitives every bb_* component builds on: the portable error type, the canonical clock, run-exactly-once, a contention-instrumented lock, byte-order helpers, memory accounting, and the reboot-reason codec.
<!-- END bbtool:brief -->
