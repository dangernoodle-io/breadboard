# button

Button input primitives — `bb_button` (the portable debounced-button
primitive) plus its GPIO backend (`bb_button_gpio`) and event-fanout helper
(`bb_button_events`), each registering with `bb_button` via the
backend-dispatch convention rather than being a standalone
consumer-facing API.
